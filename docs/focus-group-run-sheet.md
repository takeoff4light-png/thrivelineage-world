# Virtual Focus Group Run Sheet (MVP) — Notes Only (Unrecorded)

## Recording policy (locked)
- Audio/video/screenshare recording: NO
- Screenshots/screen recordings by participants: NO (requested)
- Facilitator data capture: written notes ONLY
- Notes use: product improvement + anonymous summary
- Personal identifiers in notes: avoid; use session codes (FG01, FG02...)

## Purpose (pick ONE primary)
- [ ] Concept clarity
- [ ] Trust/ethics of "guaranteed exposure"
- [ ] Workflow comprehension (draft -> approval -> scheduling -> posting)
- [ ] Pricing/tier expectations

## Success criteria (choose 2–3)
- [ ] 80% can explain the product back in 30 seconds
- [ ] 70% understand "guaranteed exposure" without assuming sales/virality
- [ ] Clear top 3 objections identified + replacement wording captured

## Session logistics
- Duration: 60–90 min
- Participants: 6–8
- Incentive: ______________________
- Note-taking method: bullet notes + 10-min debrief summary

## Consent script (read aloud)
"Welcome, and thank you. This is a private prototype preview.
We are not recording audio or video, and we ask that you don’t record or screenshot either.
I’ll take written notes only, and I’ll summarize feedback anonymously to improve the product.
Participation is voluntary—you can skip any question or leave at any time."

## Confidentiality (light)
"Please keep what you see here confidential: no sharing links, screenshots, or details outside this session."

## Script outline
1) 2 min: context + consent
2) 5 min: what problem do you think this solves?
3) 10 min: show the concept + ask for re-explanation
4) 15 min: test 'guaranteed exposure' wording (what does it mean to you?)
5) 15 min: test workflow understanding
6) 10 min: pricing expectations + trust factors
7) 5 min: final thoughts + thank you

## Output (after session)
- Top 3 confusions
- Top 3 trust concerns
- Best replacement phrases for "guaranteed exposure"
- Next iteration list (max 5 items)
- One-sentence positioning that participants naturally used
